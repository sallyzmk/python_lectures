# 인공지능
- 인공지능 > 머신러닝 > 딥러닝
- 머신러닝 알고리즘 (~339p)
  + 선형회귀, 결정트리
  + 결과에 대한 해석 요구
  + 통계 분석을 같이
  + 정형데이터(=엑셀 데이터, 테이블)

- 딥러닝 알고리즘 (340p~)
  + 인공신경망 알고리즘
  + 이미지(=영상인식), 자연어(=음성인식)
  + 높은 성능 요구
